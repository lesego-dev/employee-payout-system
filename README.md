# Employee Payout Management System

## Overview

This project demonstrates an object-oriented system for calculating employee payouts based on different compensation models.

The system supports:

- Salary employees
- Hourly employees
- Hybrid employees

## Features

- Object-oriented design using classes
- Dynamic payout calculation
- Bonus calculation based on performance
- Clean and extensible architecture

## Tech Stack

- JavaScript
- Node.js
- Object-Oriented Programming (OOP)
